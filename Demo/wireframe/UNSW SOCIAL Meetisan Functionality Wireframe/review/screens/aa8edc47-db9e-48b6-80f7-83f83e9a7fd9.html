Page Description:
This page shows a menu in the person's tag screen

All information, except for the pop up menu, should be disabled and greyed out

Interactions:
> Press "Select All" to select all of the person's tags
> Press "Unselect All" to unselect all of the person's tags
> Press "Cancel" to go back to the person's tag page

When tags are selected:
> Press "Search for People with Selected Tags" to redirect to the 'Meet - People (Closest)' screen. These selected tags should overwrite existing the tag filter
> Press "Search for Meetings with Selected Tags" to redirect to the 'Meet - Meeting (Closest)' screen. These selected tags should overwrite existing the tag filter
