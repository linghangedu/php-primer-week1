Page Description:
This page allows the user to edit details of their meeting. This page should only be accessible if the user created the meeting.

All information should be pre-populated by input information when Creating the meeting.

Interactions:
> Press "Search for People with these Meeting Tags" to go to the 'Meet - People (Closest)' screen. The meeting's tags should be automatically applied as filters when viewing People and Meetings
> Press "Search for Meetings with these Meeting Tags" to go to the 'Meet - Meetings (Closest)' screen. The meeting's tags should be automatically applied as filters when viewing People and Meetings

> If the user created this event, press "Edit Meeting" to redirect to the "Edit Event" screen

> Press "Cancel" to go back to the Meeting Details page
