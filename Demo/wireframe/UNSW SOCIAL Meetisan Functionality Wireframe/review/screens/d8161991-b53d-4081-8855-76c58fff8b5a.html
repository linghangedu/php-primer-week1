Page Description:
This page allows the user to Create an event.

Interactions:
> Press "Meeting Tags" to go to the 'Add Tags' screen
> Press "Invite People" to see a list of people you can invite to the meeting
> Press the privacy switch to change the meeting to Private/Public
> Press the "Pin exact location on map" to go to the 'Create - Map Location' screen
> Press the blue Time, Date, and Duration buttons to edit
> Press the image placeholder to go to the 'Create - Add Picture' screen
> Press "Let's Meet" to create the meeting

Input text for:
> Meeting title
> Meeting description
> Meeting location
> How to find the attendees

Restrictions:
> Can only select 3 tags per meeting
> A user can only begin inviting people after at least 1 tag has been chosen for the meeting
> A meeting can only be published if each field has been completed
